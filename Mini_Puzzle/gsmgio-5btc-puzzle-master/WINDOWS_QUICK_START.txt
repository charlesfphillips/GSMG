TROUBLESHOOTING:

Problem: "Python not found"
Solution: 
  - Install Python from python.org
  - Check "Add Python to PATH" during installation
  - Restart Command Prompt

Problem: "OpenSSL not found"
Solution:
  - Install from https://slproweb.com/products/Win32OpenSSL.html
  - Choose "Add OpenSSL to PATH" during installation
  - Restart Command Prompt

Problem: "salphaseion.html not found"
Solution:
  - Save the HTML file in the same folder as the scripts
  - Name it exactly: salphaseion.html
  - Make sure it's not salphaseion.html.txt

Problem: "All passwords failed"
Solution:
  - Check that ABBA decoding produced readable text
  - Check that HEX decoding produced readable text
  - Try different combinations in 04_combine_passwords.py
  - Check the hex/ABBA decoding by opening:
    - abba_decoded.txt
    - hex_decoded.txt
