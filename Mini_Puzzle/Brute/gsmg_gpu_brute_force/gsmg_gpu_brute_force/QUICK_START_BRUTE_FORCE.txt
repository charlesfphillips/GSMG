# QUICK START GUIDE - GPU Brute Force

## 1. Setup (60 seconds)

```bash
chmod +x setup_brute_force.sh
./setup_brute_force.sh
```

## 2. Run (minutes to hours)

```bash
python3 optimized_gpu_brute_force_new.py
```

## 3. Wait for Results

The script will test ~25 million permutations across 7 smart character subsets.

**Estimated Time:**
- With GPU: 5-15 minutes
- Without GPU: 20-60 minutes

## What It Does

Tests mappings from puzzle alphabet `abcdefghio` to Base58 characters to find:
- k1: 51-char WIF key starting with '5'
- k2: 52-char WIF key starting with 'K' or 'L'

## Troubleshooting:

**"No module named 'numba'"**
```bash
pip3 install numba numpy
```

**Takes too long**
- Let it run! 25M permutations takes time
- GPU speeds it up 10-100x
- Progress updates every few seconds

**No results found**
- The mapping might not be in the smart subsets
- May need exhaustive search (much slower)
- Or a different decoding method is required
